Assimp for Openb3d

Assimp mesh loader and a few helper functions. To build Assimp use bmk makemods -a -w -g x64 openb3dlibs.assimp. Requires BaH.Boost source and Koriolis.Zipstream module binaries, see source links below. To get BaH.Boost with Subversion open command-line and be sure you "cd" to bah.mod before checkout.
